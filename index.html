TinyCMS by franktudor

TinyCMS Database-less Coldfusion Based Content Management System

This opensource application was developed because there is a need and niche for database-less CMS application. We have lots of neat things built right in. The code was developed in wonderful language called Coldfusion. Give it a run and enjoy. Purpose Development is getting increasingly complicated plus with the red sun rising on CSS3 and HTML5 development it is only going to get increasingly complicated, why not eliminate the by-product learning curves and just focus on client side development. I developed this for me to whip out CSS sites and content quickly and it may not have all the bells and whistles you need... so if you are looking for an additional feature drop me a line.

Dependencies

Coldfusion

Install

Install Coldfusion Developer and copy the unzipped folder to the wwwroot (where every it might be). Make sure Coldfusion is running then navigate to your localhost/tinyCMS to see the results. Tack on /toolkit and get into the admin section using username/password (as the user and password) :)
